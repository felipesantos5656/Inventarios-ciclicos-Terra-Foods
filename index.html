☁ Configurar guardado en la nube

Para que los cambios se guarden automáticamente y todos vean lo mismo, necesitas una cuenta gratuita en JSONBin.io:

  1. Ve a jsonbin.io → crea cuenta gratuita
  2. Click "Create Bin" → en el contenido escribe: {"groups":null,"state":null} → Save
  3. Copia el BIN ID (número largo en la URL después de /b/)
  4. Ve a API Keys (menú izquierdo) → crea una Master Key → cópiala
  5. Abre este archivo HTML en un editor de texto, busca las líneas:
    const JSONBIN_BIN_ID = "69d309adaaba882197ca5019";
    const JSONBIN_API_KEY = "$2a$10$x9Sen.QNpmze4OeEF.QcY.Jgz8BKYvWJ0zqUEdPVIOd0x6zIVs/8O";
  6. Pega los valores entre las comillas y guarda el archivo. ¡Listo!
✓ Gratis hasta 10.000 requests/mes
✓ Todos los que tengan el link ven los cambios en tiempo real
